what is css ?

  1. css stands for Cascading Style Sheet.
  2. css describes how HTML Elements are to be displayed on screen,paper or in other media.
  3. It can control the layout of multiple web pages all at once.
  4. External Stylesheet are stored in CSS files.

CSS can be Added To HTML Elements in 3 Ways:

  • Inline-by using the style attribute inside HTML elements.
  • Internal -by using a style elememt in the head section
  • External-by using a link element to link to an external css file.
  • The most common way to add css ,is to keep the styles in external css files.However ,because this is easier to demostrate and easier for you to try it yourself.

When comparing css class vs ID the difference is that css class applies a style to multiple elements.ID,on the other hand ,applies a styles to one unique elemnt.ID is also special in that you can use a special URL to link directly to an element and its used by Javascript.

In CSS, selectors are used to target a specofic element or range of elements on a web page.Once an element has been targeted ,a style or set of of styles can be applies to the element.

There is a wide range of selectors available.two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
